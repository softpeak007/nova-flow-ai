# Nova Flow AI

AI-powered CRM, workflow automation, WhatsApp campaign, funnel builder, and business automation dashboard.

## Live Demo

https://nova-flow-ai-six.vercel.app/

## Project Overview

Nova Flow AI is a modern SaaS-style automation platform built for businesses that want to capture leads, qualify customers, manage CRM pipelines, automate WhatsApp follow-ups, build funnels, and control AI agents from one dashboard.

## Key Features

- AI-powered CRM dashboard
- Lead capture and lead management
- Visual workflow builder
- WhatsApp broadcast campaign system
- AI agent playground
- Funnel builder
- Auto-calling dialer mockup
- Course hosting / LMS module
- Community forum module
- Calendar booking system
- Billing and subscription panel
- Developer API and Supabase connection panel

## Tech Stack

- React
- Vite
- JavaScript
- HTML
- CSS
- Vercel Deployment
- Supabase-ready architecture
